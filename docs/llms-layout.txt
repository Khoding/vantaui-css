# VantaUI CSS: Layout & Utilities

> Containers, flex composites, grids, breakpoint show/hide, the full utility-class catalogue, HUD primitives (type roles, plates, brackets, backdrops, status dot), and the prose layout for long-form text. All free-floating helpers in this file use the `vui-` prefix.

Prerequisites: opt in with `class="vui"` on a root element (usually `<body>`). Bare semantic HTML is styled with NO prefix; the `vui-` prefix marks free-floating layout/utility/primitive helpers (collision-safe). All VantaUI styles are zero-specificity (`:where()`) inside cascade layers, so your own unlayered classes always win. You should never use `!important`. ## Breakpoints (mobile-first, min-width)

**s** 36rem/576px Â· **m** 48rem/768px Â· **l** 64rem/1024px Â· **xl** 80rem/1280px. Default to mobile styles; opt into larger layouts upward. Tokens: `--bp-{s,m,l,xl}`.

## Container

- `vui-container`: centered, max-width `--container-max`, inline padding `--container-pad`.
- `vui-container--narrow`: max-width `--container-narrow`.
- `vui-container--fluid`: has no max-width.

## Content section (`vui-section`)

A titled content band: a leading hairline divider, generous block padding, and a scroll anchor for in-page links. Stacked sections read as one rhythmic document (the first drops its divider). It sets no `display`, so compose it with `vui-prose` (for a readable measure) or `vui-stack` (for child rhythm). A child heading is the title; a `<p class="blurb">` is the lead paragraph (muted, zero margin).

```html
<section class="vui-section vui-prose" id="philosophy">
  <h2>Philosophy</h2>
  <p class="blurb">One short framing sentence.</p>
  <!-- section content â€¦ -->
</section>
```

## Named flex composites (prefer these)

Reach for these first; the atomic flex utilities below are the escape hatch.

- `vui-row`: horizontal, centered, gap `--gutter`. Add `vui-row--wrap` to wrap.
- `vui-col`: vertical flex column.
- `vui-stack`: vertical rhythm column with a gap of `--space-5`.
- `vui-cluster`: wrapping horizontal group with a gap of `--space-4`.
- `vui-center`: grid place-items center.
- `vui-between`: space-between row, centered.
- `vui-spacer`: `flex: 1 1 auto` (pushes siblings apart).

## Auto grid (zero-config, responsive)

- `vui-autogrid`: cards flow and wrap; control the min track with `style="--vui-min: 16rem"`. Add `vui-autogrid--fit` to stretch the last row.

```html
<div class="vui-autogrid" style="--vui-min: 16rem">â€¦ cards â€¦</div>
```

## 12-column grid

`vui-grid` wraps children that carry span classes. Children span full width by default (stack on mobile) and opt into spans at each breakpoint up: `vui-s1â€¦vui-s12`, `vui-m1â€¦vui-m12`, `vui-l1â€¦vui-l12`.

```html
<div class="vui-grid">
  <article class="vui-s12 vui-m6 vui-l4">â€¦</article>
  <article class="vui-s12 vui-m6 vui-l8">â€¦</article>
</div>
```

## Show / hide (hide-only, preserves natural display)

- `vui-until-{s,m,l,xl}`: hidden at or above the breakpoint.
- `vui-from-{s,m,l,xl}`: hidden below the breakpoint.

## HUD primitives (base building blocks, `vui-` prefixed)

The vocabulary every component is built from is also useful on its own.

- **Type roles**: `vui-display` `vui-title` `vui-heading` `vui-mono` `vui-readout` (big numeric, HUD font) `vui-label` `vui-eyebrow` (renders a `//` prefix) `vui-signal` (glowing cyan text).
- **Plates / frames**: `vui-chamfer` (cut top-left + bottom-right), `vui-notch` (all four corners cut), `vui-brackets` (L-shaped corner ticks), `vui-rule` (hairline with fading center). `vui-chamfer`/`vui-notch` clip their paint AND contain inline overflow (`overflow-x: clip`).
- **Panel surface** (`vui-panel`): full panel look (background, border, shadow, chamfer/corner-shape, `overflow-x: clip`) on any element. Use when `<article>` is not possible, such as when a `<router-link>` renders as `<a>`. Tones like `cyan`, `amber`, `red`, and `green` remap `--_c` so child components pick up the accent. Add `linked` for hover interaction (shifting the border and background). Add `vui-clip` to opt out of the corner-shape upgrade (preserving the open, unstroked diagonal). You can compose freely with layout utilities, since your own classes will override the block/flex display.

```html
<!-- link that looks like a panel; router-link renders as <a> -->
<router-link class="vui-panel cyan linked" to="/stats">
  <small class="vui-eyebrow">Total Ops</small>
  <div class="vui-readout">1 204</div>
</router-link>
```
- **Textured backdrops**: `vui-bg-grid` `vui-bg-scan` `vui-bg-hud` (ink + grid + scanlines + vignette).
- **Status dot**: `vui-dot` (default online/green, pulsing) + `vui-dot--warn` `vui-dot--threat` `vui-dot--cyan` `vui-dot--idle`.
- **Labelled divider** (in a `.vui` scope): `<div class="divider" data-label="SECTOR">` draws a centered, uppercase label between two fading hairlines; without `data-label` it's a plain hairline divider.

## Prose layout (`vui-prose`) for long-form articles

A grid that centers text to a readable measure (`--prose-width`, default 65ch) while letting images, quotes, `<pre>`, and `<figure>` bleed wider (`--bleed-width`, default 85ch). Direct children land in the content column by default.

- Bleed wider: `vui-bleed`, `bleed`, or `half-bleed` (elements like `<img>`, `<figure>`, `<blockquote>`, and `<pre>` bleed automatically). A lone-image paragraph (`<p><img></p>`, common from Markdown) bleeds too.
- Edge-to-edge: `vui-bleed-full` / `bleed-full`.
- Blockquotes get an accent rail; figures style their `<figcaption>` in mono.

```html
<article class="vui-prose">
  <h1>Mission Brief</h1>
  <p>Readable measureâ€¦</p>
  <figure><img src="map.png" alt=""><figcaption>Sector map</figcaption></figure>
  <blockquote>Pulled-in accent quote.</blockquote>
</article>
```

Note: `vui-prose` is a layout, so apply it to an `<article>` that is NOT a panel. Since `.vui article:not(.vui-prose):not(.prose)` is the panel selector, the prose class opts the article out of panel styling.
